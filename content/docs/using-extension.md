This guide covers everything you need to know about using the Stash extension effectively.

## Sharing Tabs

### Basic Sharing

1. **Open tabs** you want to share
2. **Select multiple tabs** using:
   - **Cmd+Click** (Mac) or **Ctrl+Click** (Windows/Linux) for individual selection
   - **Shift+Click** to select a consecutive range
3. **Right-click** on any selected tab
4. Choose **"Share selected tabs…"** from the context menu

### What Happens When You Share

When you share tabs:
- A share link is automatically copied to your clipboard
- You'll see a notification confirming the action
- The link contains all selected tab information encoded within it

## Multi-Select Techniques

### Selecting Specific Tabs
Use **Cmd+Click** (Mac) or **Ctrl+Click** (Windows/Linux) to hand-pick specific tabs from your browser window.

### Selecting Ranges
Use **Shift+Click** to select all tabs between two clicked tabs. This is useful for selecting consecutive tabs like all tabs in a project.

### Deselecting Tabs
Click any already selected tab to remove it from your selection.

## After Sharing

Once you create a share link:

1. **Notification**: You'll see "Link copied!" notification
2. **Clipboard**: The link is ready to paste anywhere
3. **Recipients**: Anyone with the link can view your tabs on any device

### Link Details
- Link format: `https://stash.illo.fyi/s/#p=...`
- Contains all selected tab URLs and titles
- Works on desktop and mobile browsers
- No app installation needed to view shared tabs

## Tips and Best Practices

### Selecting the Right Tabs
- Include only relevant tabs for your audience
- Consider the URL length limit (roughly 50-100 typical tabs)
- Remove sensitive or personal tabs before sharing

### Organizing Tabs for Sharing
- Group related tabs together
- Close unnecessary tabs before creating a share link
- Use descriptive tab names for better context

## Troubleshooting

### Common Issues

**"No tabs selected"**
- Make sure you've highlighted at least one tab
- Try right-clicking on a selected tab instead

**"Link too long" error**
- Reduce the number of selected tabs
- Remove unnecessary tabs from your selection

**Extension not working**
- Refresh the extension in browser management
- Check that Stash has necessary permissions
