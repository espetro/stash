Stash provides several customization options to match your preferences and workflow. Access settings through the extension popup or browser extension management.

## Link Expiry Settings

Control how long your shared links remain active:

### 24 Hours
- Use for time-sensitive information
- Links automatically expire after one day
- Good for temporary research or discussions

### 7 Days  
- Great for weekly projects or ongoing work
- Links work for one full week
- Useful for team collaboration sessions

### 30 Days
- Extended sharing for long-term projects
- Links remain active for one month
- Ideal for documentation or reference collections

### Never
- Links never expire (default setting)
- Perfect for permanent references
- No automatic expiration cleanup needed

## Appearance Settings

### Theme Options
Choose how Stash looks across all interfaces:

- **Light**: Clean, bright interface
- **Dark**: Dark mode for comfortable viewing
- **System**: Matches your operating system theme

Theme preferences sync across your devices when logged into your browser account.

## Viewer Server Configuration

### Custom Viewer URLs
By default, Stash uses `https://stash.illo.fyi` to render shared tabs. You can configure a custom viewer URL if:

- The default server is temporarily unavailable
- You want to use your own self-hosted instance
- You need integration with specific infrastructure

### Setting Custom Viewer URL
1. Open extension settings
2. Navigate to "Viewer Server" section  
3. Enter your custom URL (e.g., `https://your-domain.com`)
4. Click "Save" to validate and apply

**Important**: Custom URLs must use HTTPS for security reasons. The extension will validate the URL format before saving.

## Extension Popup Features

The Stash popup provides additional functionality:

### Tab Selection Interface
- Visual tab list with checkboxes
- Search and filter options for many tabs
- Select all/deselect all controls
- Real-time tab count display

### History Management
- View recently shared links
- Quick re-copy of previous links
- Expiry status indicators
- Clean interface for managing past shares

### Action Buttons
- **Create Link**: Generate share link from selected tabs
- **Copy Link**: Copy existing share link to clipboard
- **Settings**: Access detailed configuration options
- **History**: View sharing history

## Minimal Design Philosophy

Stash is intentionally minimal—no complex settings or configuration options. The focus remains on:

- **Speed**: Quick tab selection and sharing
- **Simplicity**: No learning curve required  
- **Privacy**: No unnecessary data collection
- **Reliability**: Core functionality that just works

If you find you need more complex features, Stash may not be the right tool for your use case. The extension excels at its core mission of simple, private tab sharing.

## Settings Management

### Accessing Settings
- **Via popup**: Click the gear icon ⚙️ in the extension popup
- **Via browser**: Navigate to `chrome://extensions/` (Chrome) or `about:addons` (Firefox)

### Settings Sync
Settings sync across your devices when using browser sync:
- Expiry preferences
- Theme selections  
- Viewer server configuration
- Extension permissions

### Resetting Settings
To reset to defaults:
1. Open extension settings
2. Look for "Reset to Defaults" option
3. Confirm the reset action

Most users find the default settings work well for their needs—customization options are provided for power users and specific use cases rather than everyday functionality.
